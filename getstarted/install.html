Installation

For Creating your UI you have to install the CSS from CSS UI Galaxy Component for that you have two ways First one is you can Copy paste this below link in your html head tag or you can import this link to your css file . To know more about this UI Go Through this Documentation

For HTML file 

        <link rel="stylesheet" href="https://css-ui-galaxy-c.netlify.app/CSS/base.css">

Demo

This is the demo or exmple.Use this link in your html page head tag as shown below. 


        <!DOCTYPE html>
        <html lang="en">
          <head>
        <meta charset="UTF-8">
          <meta http-equiv="X-UA-Compatible" content="IE=edge">
          <meta name="viewport" content="width=device-width, initial-scale=1.0">
          <title>Demo ui usage</title>
          <link rel="stylesheet" href="https://css-ui-galaxy-c.netlify.app/CSS/base.css"></head>
          <body>

          </body>
          </html>
For CSS file 

        @import url(https://css-ui-galaxy-c.netlify.app/CSS/base.css")
